HƯỚNG DẪN SỬ DỤNG FOLDER MUSIC
================================

Để trang web có thể phát nhạc, bạn cần thêm các file nhạc vào folder này với tên như sau:

1. song1.mp3 - Bài hát đầu tiên
2. song2.mp3 - Bài hát thứ hai  
3. song3.mp3 - Bài hát thứ ba
4. song4.mp3 - Bài hát thứ tư
5. song5.mp3 - Bài hát thứ năm

LƯU Ý:
- Chỉ hỗ trợ định dạng MP3
- Tên file phải chính xác như trên
- Kích thước file không quá lớn để tải nhanh
- Nên sử dụng nhạc có chất lượng tốt nhưng không quá nặng

GỢI Ý NHẠC CHO NGÀY 20/10:
- Nhạc tình cảm, nhẹ nhàng
- Nhạc acoustic, ballad
- Nhạc không lời
- Nhạc Việt Nam hoặc quốc tế đều được

Sau khi thêm nhạc, refresh trang web để có thể phát nhạc!
