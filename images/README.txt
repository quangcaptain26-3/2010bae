HƯỚNG DẪN SỬ DỤNG FOLDER IMAGES
================================

Để trang web hiển thị ảnh đẹp, bạn cần thêm các file ảnh vào folder này với tên như sau:

ẢNH GALLERY (4 ảnh):
1. photo1.jpg - Ảnh đầu tiên trong gallery
2. photo2.jpg - Ảnh thứ hai trong gallery  
3. photo3.jpg - Ảnh thứ ba trong gallery
4. photo4.jpg - Ảnh thứ tư trong gallery

ẢNH CẶP ĐÔI:
5. couple.jpg - Ảnh cặp đôi lớn (tỷ lệ 9:16 dọc)

ẢNH KỶ NIỆM (3 ảnh):
6. memory1.png - Kỷ niệm đầu tiên
7. memory2.jpg - Kỷ niệm thứ hai
8. memory3.jpg - Kỷ niệm thứ ba

LƯU Ý:
- Hỗ trợ định dạng: JPG, JPEG, PNG
- Kích thước khuyến nghị:
  + Gallery photos: 300x300px hoặc tỷ lệ vuông
  + Couple photo: 300x533px hoặc tỷ lệ 9:16 (dọc)
  + Memory photos: 180x320px hoặc tỷ lệ 9:16 (dọc)
- File không quá nặng để tải nhanh
- Nên sử dụng ảnh chất lượng cao, rõ nét

GỢI Ý ẢNH:
- Ảnh kỷ niệm đẹp của hai người
- Ảnh chụp cùng nhau
- Ảnh từ các dịp đặc biệt
- Ảnh selfie, ảnh du lịch

Sau khi thêm ảnh, refresh trang web để xem ảnh mới!
